个性化配置

按你的习惯，精调每一处细节

扩展选项页集中管理 AI 分组、提示词与性能策略。接口地址、模型、阈值、休眠白名单都可以自由设定，全部存储在本地浏览器，不经过任何第三方服务器。

  • 标签休眠 — 空闲时长、白名单域名，按需释放内存

安装与配置

当前为开发者加载方式，暂未上架 Chrome 应用商店。

  1. 1

    克隆仓库

    git clone git@github.com:Michaelooo/chrome-side-tabs.git
  2. 2

    安装依赖并构建

    npm install && npm run build
  3. 3

    加载到 Chrome

    打开 chrome://extensions，开启右上角「开发者模式」，点击「加载已解压的扩展程序」，选择项目根目录下的 dist/ 文件夹。

  4. 4

    个性化配置（可选）

    右键扩展图标 →「选项」，或从侧边栏进入设置页。可配置 AI 接口与模型、追加自定义分组提示词、调整自动分组阈值，以及标签休眠的空闲时长与白名单。不填 API Key 则默认按域名分组。
