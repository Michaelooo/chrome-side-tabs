# 截图与演示资源

把真实截图或录屏放进本目录，**保持以下文件名**，页面无需改动即可显示。
在资源就位前，页面会用 CSS 占位（命令面板为「即将推出」样式，配置区为选项页预览），不会出现裂图。

| 文件名 | 用途 | 建议比例 |
| --- | --- | --- |
| `hero.png` | 首屏 Hero 右侧主视觉（侧边栏整体界面） | 约 4:3 / 竖向侧栏 |
| `整理标签.mp4` | 演示区：AI 智能分组效果（自动播放、静音循环） | 约 4:3 |

- 命令面板与个性化配置演示当前为 **纯 CSS 占位**，无需额外截图。
- 格式用 PNG / MP4；宽度建议 ≥ 1000px 以保证清晰度。
- 如需换文件名，记得同步修改 `docs/index.html` 中对应引用。
